# CV

The CV is compiled automatically, and can be found in the [latest release](https://github.com/teofr/CV/releases/latest).

El CV se compila automáticamente, y se puede descargar desde el [último release](https://github.com/teofr/CV/releases/latest).


### Locally / Localmente

For English  `make en`

Para Español `make es`

For both `make all`

Para ambos `make all`
